1、首页设置了多个广告位，开奖内页也有广告位，支持多种广告类型（首页轮播图、侧边栏广告、导航广告等），可设置广告展示时间、位置、状态，支持图片广告和链接跳转；
2、后台可以添加需要采集的彩种等，支持多种采集源（168接口、第三方接口、系统采集），Windows系统+Linux系统双端都可执行采集命令；
3、支持私彩开奖数据管理、私彩开奖号码录入和修改等；
4、提供标准化的开奖数据API接口，RESTful API设计，支持JSON数据格式访问、跨域访问；
5、自动采集和发布彩票相关文章；

目前手机端所有显示的彩种数据都是基于系统采集的，也就是直接读取数据库的内容，后台修改开奖号码手机端会同步显示修改后的号码，PC端除了系统彩是实时系统数据，部分接口还是直接调用了168开奖接口，没有和系统彩一样独立出来

系统彩可以增加的类型：时时彩、快3、PC28、PK10、六合彩

采集方式最开始用的是workerman，由于效率的问题又改成了系统命令的方式，现在win系统可以通过bat方式启动采集，Linux系统可以通过sh或者php命令启动
