<div id="sc_fankui">测试反馈</div>
测试环境：Linux系统CentOS7.6、宝塔面板、Nginx、nodejs V16.2.0、MySQL5.7，运行目录不用选择，伪静态不用选择，建议开启SSL

以为是完整的，跑了一下发现有些页面的js丢失了，home、main等主页面js丢失严重，但是大部分原有的按钮是可以正常点击的，只是UI乱了（也有可能是我不会玩）

修复了一些小bug，游戏页面大部分都正常，除了第一张截图不是测试的时候截的，其他几张都是测试截的图，感兴趣的可以去修复一下

前端用了ejs模版引擎，也就是用js渲染成了html，但我感觉有点像vue，写法和路径很相似

整套像是仿的Big Mumbai这个程序，官方用的是vue写前端，而且这个我没有找到后台怎么登录。。。

搭建也不麻烦，宝塔创建一个Node项目，将文件传上去解压，选择对应的目录，直接保存就行，添加之前一定要先创建好数据库并导入

数据库修改：.env 和 /src/config/connectDB.js 文件里面将 wukong_caipiao1 改成你自己的，或者你自己设置账号密码都为 wukong_caipiao1 即可
